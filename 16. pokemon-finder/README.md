# Pokémon Finder

## Description
- A simple web application that allows users to search for a Pokémon by name and view its image and types.

## Features
- Search for any Pokémon by name.
- Displays the Pokémon's front sprite.
- Shows all types associated with the Pokémon.
- Alerts the user if the Pokémon is not found or if no input is provided.

## Technologies Used
- HTML
- CSS
- JavaScript
- [PokéAPI](https://pokeapi.co/) for fetching Pokémon data

## How to Run
1. Clone or download the repository.
2. Open `index.html` in your preferred web browser.
3. Make sure you have an active internet connection to fetch data from the PokéAPI.

## Usage
1. Enter a Pokémon name in the input field.
2. Click the "Fetch" button or press the "Enter" key.
3. The Pokémon's image and types will appear below the input.
4. Example:
   - Input: `pikachu`
   - Output: Displays Pikachu's image and types `electric`
