# Text-to-Voice Converter

## Description
- A simple web application that converts written text into spoken words using the browser's speech synthesis API. Users can select different voices for speech output.

## Features
- Converts text input to speech.
- Provides a dropdown menu to select available voices.
- Allows users to change voices dynamically.
- Stops previous speech when switching voices.

## Technologies Used
- HTML
- CSS
- JavaScript (Web Speech API)

## How to Run
1. Download or clone the project folder.
2. Open `index.html` in a modern web browser that supports the Web Speech API (e.g., Chrome, Edge, Firefox).

## Usage
1. Type your text into the textarea.
2. Select a voice from the dropdown menu.
3. Click the "Start" button to hear the text spoken.
4. Example:
   - Input: `"Hello, my name is Raven."`
   - Output: The browser speaks the text aloud in the selected voice.
