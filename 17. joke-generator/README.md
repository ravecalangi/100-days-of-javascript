# Random Joke Generator

## Description
- A simple JavaScript web application that fetches and displays random jokes from an external API when a button is clicked.

## Features
- Fetches random jokes from the JokeAPI
- Supports both single-line and two-part jokes
- Filters out NSFW, political, religious, racist, sexist, and explicit content
- Updates the UI dynamically without reloading the page
- Button-triggered joke generation

## Technologies Used
- JavaScript (ES6)
- Fetch API
- HTML DOM Manipulation
- JokeAPI (v2)

## How to Run
1. Download or clone the project files.
2. Open the `index.html` file in any modern web browser.

## Usage
1. Click the **Get Joke** button.
2. A random joke will appear on the screen.
3. If the joke is:
   - **Single type** → displayed in one paragraph
   - **Two-part type** → setup and delivery are shown separately

**Example:**
- **Output (Single Joke):**  
  `Why do programmers prefer dark mode? Because light attracts bugs.`

- **Output (Two-Part Joke):**  
  Setup: `Why did the developer go broke?`  
  Delivery: `Because he used up all his cache.`
