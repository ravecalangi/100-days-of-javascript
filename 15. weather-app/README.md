# Weather App (OpenWeather API)

## Description
- A simple weather application that fetches and displays real-time weather data for a searched city using the OpenWeather API.

## Features
- Search weather by city name
- Displays temperature in Celsius
- Shows humidity and wind speed
- Dynamic weather icons based on weather condition
- Error message for invalid or unknown city names

## Technologies Used
- JavaScript (ES6)
- HTML
- CSS
- OpenWeather API

## How to Run
1. Download or clone the project.
2. Make sure the files are structured correctly (HTML, CSS, JS, and `img` folder for icons).
3. Open `index.html` in your web browser.

## Usage
1. Type a city name in the search input.
2. Click the search button.
3. The app will display the current weather details.
