# Contact Form with Auto-Reset

## Description
- A simple contact form webpage that allows users to submit their name, email, and message.  
- The form automatically resets a few moments after submission.

## Features
- Responsive contact form built with HTML and CSS.  
- Uses Web3Forms API for form submission.  
- Automatically clears input fields after submission.  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla)  

## How to Run
1. Clone or download the project files.  
2. Open `index.html` in your web browser.  

## Usage
1. Fill out the **Name**, **Email**, and **Message** fields.  
2. Click **Submit** to send your details via Web3Forms.  
3. The form will reset automatically after submission.
